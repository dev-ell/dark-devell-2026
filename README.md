A modern, high-contrast dark theme for Visual Studio Code, meticulously designed for 2026's development standards. **Dark Devell 2026** focuses on readability, vibrant syntax highlighting, and a premium "glassmorphism" inspired UI.

## ✨ Features

- **Deep Midnight Background**: A custom `#121314` background that reduces eye strain during long coding sessions.
- **Vibrant Syntax Highlighting**: High-contrast colors for variables, functions, and keywords to help you scan code faster.
- **Premium UI Accents**: Smooth blue accents (`#3994BC`) and subtle borders for a modern, sleek interface.
- **Semantic Colorization**: Full support for semantic highlighting in modern languages (TypeScript, Python, Rust, etc.).

## 🚀 Installation

1. Open **Visual Studio Code**.
2. Press `Ctrl+P` (or `Cmd+P` on macOS) to open the Quick Open dialog.
3. Type `ext install devell.darkdevell2026` and press **Enter**.
4. Alternatively, search for `Dark Devell 2026` in the Extensions view (`Ctrl+Shift+X`).

## 🛠 Usage

Once installed, you can activate the theme via:

1. `File` > `Preferences` > `Color Theme` (or `Cmd+K Cmd+T`).
2. Select **Dark 2026 devell** from the list.
